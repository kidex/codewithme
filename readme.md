# CODEWITHME

Bine ai venit în **CODEWITHME** - proiectul în care transformăm învățarea programării în pași simpli, practici și ușor de urmărit.

## Despre proiect

`CODEWITHME` este construit în jurul unor **sesiuni video scurte**, iar după fiecare sesiune codul scris este adăugat în foldere separate, ca să poți:

- urmări progresul pas cu pas
- reveni rapid la exemplele explicate
- învăța prin practică, nu doar teorie

## Cine dezvoltă proiectul

Acest proiect este dezvoltat de **Alexandru Marcu Academy**.
